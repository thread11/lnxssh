### Start

```
# Basic
./lnxssh
./lnxssh --help

# Advanced
./lnxssh --debug=true
```

### Access
```
http://127.0.0.1:1234/
http://127.0.0.1:1234/?host=127.0.0.1
http://127.0.0.1:1234/?host=127.0.0.1&user=root
http://127.0.0.1:1234/?host=127.0.0.1&port=22&user=root&password=123456
```
